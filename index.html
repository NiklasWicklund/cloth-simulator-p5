Cloth simulator

This is a cloth simulator that make use of Verlet Integration and The Jacobsen Method. Due to performance limitations the cloth is scaled down but a larger version was done in C++ and OpenGL. The code for that can be found here.

Verlet Integration

Verlet integration is a numerical method used to integrate Newton's equations of motion. It is a time integrator that is used to calculate the position of a particle in the next time step. The method is based on the Taylor expansion of the position of a particle.

Jacobsen Method

The Jacobsen method is used to apply constraints to the cloth. The method is based on the idea that the cloth is made up of springs that are connected to each other.
